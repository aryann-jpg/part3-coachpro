# coachpro-devops
CoachPro is a workout planning and client-management system built with Node.js, Express, and a simple modular structure. It allows fitness coaches to create workout plans, update existing routines, and use reusable templates, all stored in a JSON-based database.

CoachPro is designed to make weekly workout planning faster and more organised.
It features:

- Create Workout Plan
- Edit Workout Plan
- Workout Template creation
- Clean separation of /public/js, /utils, /models, and index.js
- JSON-stored workout data
- User-friendly frontend with alerts and modal confirmations

Running the project:
- npm install
- npm install express body-parser bcryptjs cookie-parser jsonwebtoken
- node index.js
